# 生日祝福页面（手机端可爱版）

## 直接预览
- 双击打开 `index.html` 即可本地预览（建议用手机浏览器打开/或局域网访问）。

## 个性化内容
在 `app.js` 顶部的 `CONFIG` 里改这几项即可：
- `toName`：她的名字
- `fromName`：你的署名
- `wish`：卡片上的主祝福
- `noteTitle` / `noteBody`：弹窗祝福内容
- `cards`：抽卡文案池

## 背景音乐（两种方式）
手机浏览器通常不允许“自动播放”，所以页面做成了：**必须点一次「开启背景音乐」**才会播放。

1) **放你自己的 mp3（推荐）**
- 把你的音乐文件命名为 `bgm.mp3`
- 放到 `assets/bgm.mp3`
- 再点「开启背景音乐」就会优先播放这首

2) **不放 mp3 也行**
- 页面会自动用 WebAudio 播一个轻快小旋律做背景音乐（同样需要点按钮才会响）

## 发布成链接（写入 NFC）
你需要把这个文件夹发布到一个可访问的 HTTPS 链接，然后把链接写进 NFC 芯片即可。

常见选择：
- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages
